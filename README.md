# Projets d'analyses de données et métabarcodes Flora Larreur

## Tutoriel de la pipeline DADA2

Vous trouverez ici le tutoriel DADA2 annoté et expliquéé à chaque étape via ce lien : https://github.com/floraalr/floraalr.github.io/blob/main/CC_DADA2_Flora_Larreur%20(1).md

La pipeline DADA2 prend les séquences brutes issues du séquençage d’ADN microbien et les nettoie en supprimant les séquences de mauvaise qualité, corrige les erreurs de lecture propres à la machine, puis identifie les séquences exactes d’amplicon (ASV) présentes dans l’échantillon, permettant ainsi d’obtenir une vision fine et précise de la diversité microbienne, bien plus fiable que les méthodes classiques basées sur les OTU.
